GUIDE D'INSTALLATION

1 Installer Tampermonkey

Tampermonkey est l'extension qui permet d'exécuter le script de sniping sur votre navigateur.

Chrome / Brave Microsoft Edge

2 Activer le Mode Développeur

Pour permettre l'installation de scripts externes, vous devez activer le mode développeur dans vos extensions.

Action : Allez dans chrome://extensions et cochez "Mode Développeur" en haut à droite.

3 Installer le Script ELITE

Cliquez sur le bouton ci-dessous. Tampermonkey s'ouvrira automatiquement pour vous proposer l'installation.

🚀 INSTALLER LE SCRIPT (V82.6)

Cliquez sur "Installer" dans la fenêtre Tampermonkey qui s'affiche.

4 Prêt pour le Sniping

Rendez-vous sur Shuffle.com. Le script détectera automatiquement votre compte.

  • Si votre VIP est actif sur @S_power_manager_bot, le sniper démarre.
  • Le statut passera en CONNECTED en haut de votre écran.
