GUIDE D'INSTALLATION

⚡ CONSEIL PERFORMANCE

Pour une vitesse d'interception maximale (0ms), nous recommandons l'usage de Google Chrome Canary.

Télécharger Chrome Canary

1 Installer Tampermonkey

Ajoutez l'extension Tampermonkey à votre navigateur pour gérer le script.

Figure 1 : Installation de l'extension Tampermonkey
Aller sur le Chrome Store

2 Configurer Tampermonkey

Activez le Mode Développeur dans vos extensions et autorisez l'accès aux fichiers dans les réglages de Tampermonkey.

Figure 2 : Activation du mode développeur et accès aux URL

3 Installer ELITE-CODE-HUNTER

Cliquez sur le bouton ci-dessous pour installer le script. La fenêtre de confirmation Tampermonkey s'ouvrira.

Figure 3 : Page d'installation du script ELITE-CODE-HUNTER
