Instructions Détaillées

Suivez ce guide étape par étape pour configurer votre sniper.

1 Installer Tampermonkey

Si vous avez déjà Tampermonkey, passez à l'étape suivante. Sinon, installez l'extension pour votre navigateur :

Figure: Vidéo tutoriel pour installer Tampermonkey
Chrome / Brave / Opera Microsoft Edge

2 Activer le Mode Développeur

Certains navigateurs bloquent les scripts externes. Vous devez activer le Mode Développeur :

  • Dans **Chrome/Brave**, allez sur chrome://extensions et cochez "Mode Développeur" (en haut à droite).
  • Dans **Edge**, allez sur edge://extensions et activez le "Mode Développeur" (en bas à gauche).
  • Sur la page des extensions, cliquez sur Détails sous Tampermonkey et cochez "Autoriser l'accès aux URL de fichiers".
Figure: Activation du Mode Développeur et des accès scripts

3 Installer le Script ELITE-HUNTER

Cliquez sur le lien ci-dessous. Tampermonkey affichera une fenêtre d'installation.

🚀 INSTALLER ELITE-HUNTER.user.js

1. Cliquez sur le bouton Installer dans la fenêtre qui s'ouvre.
2. Vérifiez que "ELITE-HUNTER" est bien présent dans votre tableau de bord Tampermonkey.

Figure: Fenêtre d'installation de Tampermonkey

4 Redémarrer le Navigateur

Pour garantir que le script se charge correctement, fermez toutes les fenêtres de votre navigateur et rouvrez-le.

5 Ouvrir Shuffle.com

Allez sur Shuffle.com et connectez-vous à votre compte.

  • Le script insère un panneau de contrôle **tout en bas de la page**.
  • Cliquez sur le bouton Connect pour lier le sniper au serveur.
  • Une fois que le statut affiche CONNECTED, le sniper est prêt à auto-claim les codes.
