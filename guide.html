GUIDE D'INSTALLATION

Activez votre sniper ELITE-CODE-HUNTER en quelques étapes simples.

1 Achat de la licence

Allez sur le bot @S_power_manager_bot. Après paiement, le bot vous donnera votre clé de licence unique.

2 Installation

Utilisez l'extension Tampermonkey sur votre navigateur. Importez le script fourni par le bot Manager.

3 Configuration

Sur Shuffle, ouvrez le menu ELITE et collez votre clé. Le statut doit afficher "CONNECTED (0ms)".
