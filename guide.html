    GUIDE D'INSTALLATION COMPLET

    Suivez attentivement ces étapes pour configurer votre sniper automatique ELITE-CODE-HUNTER.

    Quick Setup (Instructions Rapides)

    1. Installez Tampermonkey sur votre navigateur.
    2. Activez le Mode Développeur si nécessaire.
    3. Installez le script ELITE-CODE-HUNTER.
    4. Redémarrez votre navigateur et connectez-vous à Shuffle.com.

    1 Installer Tampermonkey

    Si vous avez déjà Tampermonkey, passez à l'étape suivante. Sinon, installez l'extension pour votre navigateur :

    Figure : Installation de Tampermonkey
    🌐 Chrome Web Store 🟦 Microsoft Edge

    2 Activer le Mode Développeur

    Certains navigateurs Chromium bloquent les scripts externes. Activez le Mode Développeur :

    Figure : Activation du Mode Développeur sur Chrome
    • Allez dans vos extensions et cochez "Mode Développeur" (Developer Mode) en haut à droite.
    • Dans les détails de Tampermonkey, activez "Autoriser l'accès aux URL de fichiers".

    3 Installer le Script ELITE-CODE-HUNTER

    Cliquez sur le lien fourni par le bot pour installer la dernière version du script.

    Figure : Fenêtre d'installation de Tampermonkey

    Une page Tampermonkey s'ouvrira automatiquement pour vous proposer de cliquer sur "Installer".

    4 Recommandation de Navigateur

    Pour une vitesse et une fluidité optimales, nous vous recommandons vivement d'utiliser Google Chrome Canary.

    🌐 Télécharger Google Chrome Canary

    5 Utilisation sur Shuffle.com

    Connectez-vous à votre compte sur Shuffle.com. Le script détecte votre pseudo Shuffle automatiquement.

    • Le statut affichera "CONNECTED (0ms)" une fois la connexion établie.

    Dépannage & Astuces

    Le script n'apparaît pas sur Shuffle : Vérifiez que Tampermonkey est bien activé et que le Mode Développeur est activé.
    Message "Abonnement Expiré" : Assurez-vous d'être connecté avec le bon pseudo Shuffle.
