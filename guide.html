Guide d'Installation Rapide

Suivez ces instructions pour configurer votre sniper automatique en moins de 5 minutes.

Quick Setup (Résumé)

  1. Installez Tampermonkey sur votre navigateur.
  2. Activez le Mode Développeur dans vos extensions.
  3. Ajoutez le script fourni par le bot : ELITE-HUNTER.user.js.
  4. Lancez Shuffle.com et laissez la magie opérer.

1 Installer Tampermonkey

Tampermonkey est le gestionnaire de scripts indispensable pour faire tourner ELITE-CODE-HUNTER. Choisissez votre navigateur :

🌐 Chrome Web Store 🟦 Edge Add-ons

2 Activer le Mode Développeur

Certains navigateurs bloquent les scripts externes. Pour autoriser le sniper :

  • Allez dans chrome://extensions (ou via le menu Extensions).
  • Cochez la case "Mode Développeur" en haut à droite.
  • Dans les paramètres de Tampermonkey, assurez-vous que "Autoriser l'accès aux URL de fichiers" est activé.

3 Installer le Script ELITE

Une fois votre abonnement validé sur le bot @S_power_manager_bot, vous recevrez un lien vers le script.

  1. Cliquez sur le lien du script (se termine par .user.js).
  2. Une page Tampermonkey va s'ouvrir. Cliquez sur le bouton "Installer" ou "Mettre à jour".
  3. Redémarrez votre navigateur pour finaliser l'injection.

4 Connexion & Auto-Claim

Allez sur Shuffle.com et connectez-vous. Le script détecte votre pseudo Shuffle automatiquement.

  • Le script injecte un panneau discret en haut de la page.
  • Cliquez sur "Connect" si la connexion ne se fait pas toute seule.
  • Une fois le statut "CONNECTED" affiché, vous êtes prêt. Le script claimera les codes dès leur apparition.

🛠️ Dépannage & Astuces

Le script n'apparaît pas sur Shuffle : Vérifiez que Tampermonkey est bien activé et que le Mode Développeur est ON. Essayez de vider le cache du navigateur.
Message "Subscription Expired" : Assurez-vous d'être connecté avec le bon pseudo Shuffle. Si vous avez changé de pseudo, contactez le support du bot.
Conseil de performance : Pour un sniping à 0ms, désactivez le "Mode Économie de mémoire" dans les paramètres de votre navigateur.
