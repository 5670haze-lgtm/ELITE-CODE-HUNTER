Guide d'Installation Rapide

Pré-requis

1. Un compte Shuffle actif.
2. Votre forfait VIP activé sur le bot @S_power_manager_bot.

1 Installer Tampermonkey

Téléchargez l'extension officielle pour votre navigateur :

🌐 Chrome / Brave 🟦 Microsoft Edge

2 Activer le Mode Développeur

Pour autoriser l'exécution du sniper, allez dans vos extensions et activez le "Mode Développeur" (Developer Mode) en haut à droite de votre navigateur.

3 Télécharger le Script ELITE

Cliquez sur le bouton ci-dessous pour installer la dernière version du script (V82.6) :

🚀 TÉLÉCHARGER LE SCRIPT (.user.js)

Note : Si Tampermonkey est bien installé, une page s'ouvrira automatiquement pour vous proposer de cliquer sur "Installer".

4 Utilisation sur Shuffle

Connectez-vous à votre compte sur Shuffle.com. Le script détecte votre pseudo. Si votre VIP est actif, le voyant passera au VERT et le sniping commencera immédiatement.

💡 Astuce Performance

Pour une vitesse maximale, laissez l'onglet Shuffle visible (ne réduisez pas la fenêtre du navigateur). Les navigateurs ralentissent les scripts dans les onglets cachés.
